## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository featuring:

- JWT authentication with passport
- User management
- A docker-based mySQL database

## Installation

```bash
$ npm install
```

## Running the app

```bash
# start database
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

#TODO
```
